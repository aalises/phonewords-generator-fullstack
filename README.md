# Phonewords Generator Full Stack

Generator of Phonewords, Full Stack.

To start fiddling with it, just run

```
npm install
```

and then 

```
npm run dev
``` 
it will open the application at port `4200` on `localhost`

to build run 
```
npm run build
```

### Frontend

The Frontend uses:

- React 16.x
- Parcel Bundler
- Kiwi.com Orbit Components
- Cypress
- Styled Components
